# steams3x

**Modern Steam games installer and manager**

Alternative to Steam Tools and etc.

[Download](https://github.com/kryyaa/steams3x/releases) • [Telegram](https://t.me/kryyaasoft)

---

</div>

## Features

- Auto game installation from multiple sources
- Automatic Steam process management
- Complete uninstall with manifest cleanup
- Repair function with auto-restore

## Installation

Download the latest release and run `steams3x.exe`

## Usage

**Install games**
```
Paste Steam URLs or App IDs (one per line)
Click "Install"
```

**Remove games**
```
Paste URLs or App IDs
Click "Remove"
```

**Repair**
```
Click "Repair" to backup, clean, and restore cracks
```

## Supported formats

- `https://store.steampowered.com/app/730`
- `https://steamcommunity.com/app/730`
- `730`

## Data sources

1. [ManifestHub](https://github.com/SteamAutoCracks/ManifestHub) (primary)
2. kernelos.org (fallback)

## Requirements

- Windows 7+
- Steam installed
- .NET Framework 4.7.2+

## Tested on
- Windows 11
- Steam client x64 / x86
